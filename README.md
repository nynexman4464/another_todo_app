# Another Todo

A toy todo app for testing UI frameworks. Built with Vite, React Router, and Astryx.

## Features

- Add todos from the main input
- Track created and updated timestamps
- Mark todos complete and keep completed items below open work
- Edit todo titles inline
- Confirm before deleting todos
- Filter the list by all, open, or closed items
- Sort by newest, oldest, or recently updated

## Development

Install dependencies:

```sh
npm install
```

Start the dev server:

```sh
npm run dev
```

Run checks:

```sh
npm run lint
npm run build
```
